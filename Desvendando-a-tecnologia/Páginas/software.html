Software

O que é o software?

O software é um conjunto de instruções que juntas ao serem executadas,fornecem funções e desempenho
desejados, descrito a grosso modo, o software é basicamente um programa de computador. O software
recebe dois nomes, sendo eles o software de sistema e o software de aplicativo, vamos abordar
primeiramente o software de sistema.

Software de sistema

O software de sistema consiste em uma coleção de programas para apoiar outros programas, realizando
tarefas de gerenciamento por exemplo, dentro dessa categoria, temos duas categorias, sendo elas o software
operacional e o software utilitário.

  • Software operacional

    • O software operacional também conhecido como sistema operacional é o software responsável
    pelo funcionamento geral dos sistemas de computação,sem eles esses sistemas não funcionariam,
    como exemplos de softwares operacionais, temos o Windows, o Linux e o MacOS. A principal função desse
    software é gerenciar os recursos do computador e controlar a execução de programas pelo processador.

  • Software utilitário

    • O Software utilitário vem com a função de auxiliar o software operacional, fazendo gerenciamento de
    Dispositivos e programas, e anti-vírus por exemplo. Alguns exemplos de Softwares utilitários conhecidos:
    Avast, AVG, CCleaner entre outros.

Software aplicativo

O Software aplicativo mais conhecido como app, surge para ajudar o usuário ao invés de auxiliar o sistema
operacional, fazendo tarefas específicas como por exemplo, editar um video, criar uma planilha
ou navegar na internet, alguns exemplos que posso citar são: Sony vegas, excel e google chrome.
